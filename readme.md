## Инструкция
Развернуть бота по следующим шагам:
- Установите Git на свой компьютер, если у вас еще его нет. Вы можете скачать и установить Git с официального сайта: [Git](https://git-scm.com/downloads)
- Откройте командную строку или терминал и перейдите в папку, в которой вы хотите сохранить питон-файл.
- Склонируйте репозиторий с GitHub с помощью команды `git clone URL`, где `URL` - это ссылка на репозиторий. 
- Перейдите в папку репозитория, которую вы только что склонировали, с помощью команды `cd repository`.
- Установите все зависимости, если они указаны в файле `requirements.txt`. Вы можете сделать это с помощью команды `pip install -r requirements.txt`.
- Теперь вы можете запустить питон-файл, введя имя файла: `main.py`.

 В настоящие время бот запущен на сервере через платфрму[pythonanywhere](https://www.pythonanywhere.com)
